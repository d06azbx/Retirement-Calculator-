# Retirement Planner with ML
A financial tool using **Linear Regression** to predict wealth accumulation.

### How it works:
- **ML Model:** Uses Scikit-Learn Linear Regression to calculate growth trends.
- **Inputs:** User-defined asset allocation and inflation.
- **Data:** Benchmarked against Nifty 50 (12%) and Midcap (15%) historical returns.

### Setup:
`pip install -r requirements.txt`
`streamlit run app.py`
